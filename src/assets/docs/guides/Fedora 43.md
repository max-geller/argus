# Fedora 43

## Overview

Fedora 43 is a modern, cutting-edge Linux distribution developed by the Fedora Project and sponsored by Red Hat. Released in late 2023, Fedora 43 continues Fedoraâ€™s tradition of delivering the latest open-source technologies with a strong focus on innovation, security, and ease of use.

This release features:

- **GNOME 45**: The latest GNOME desktop environment, bringing visual improvements and enhanced productivity features.
- **Updated software stack**: Ships with updated development tools, GCC, Python, and the newest stable software packages.
- **Improved hardware support**: Includes enhanced drivers and better out-of-the-box compatibility with recent hardware.
- **Security enhancements**: Incorporates upstream security updates and improvements to SELinux and firewalls.
- **Wayland by default**: Wayland is used as the default graphics stack for better performance and security.

Fedora 43 targets developers, enthusiasts, and users who prefer leading-edge technology, while maintaining a commitment to stability and open-source principles. It is available in Workstation, Server, and IoT editions, each tailored for different use cases. Fedora also serves as an upstream source for Red Hat Enterprise Linux, making it an important part of the Linux ecosystem.
