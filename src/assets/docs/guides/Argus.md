# Argus Guide

Argus is the Settings, Documentation, and Version Management hub for MaxOS. It is a modern, easy-to-use, and efficient desktop environment built with Tauri, Angular, and Angular Material.

## Key Features

- Settings Management for MaxOS
  - Hyprland Configuration
  - Waybar Configuration
- Access Documentation
  - Cheat Sheets
  - Guides
  - Keybinding Maps
- Version Management
  - Plan New Releases
  - Manage Current Release
  - Task Management
