# Daily Ops Checklist

## Before You Start
1. Launch Argus control center (`Super + H`).
2. Ensure Waybar, Wofi, and notifications are running.

## Workflow Template
- Workspace 1: Browser + Notes
- Workspace 2: Code
- Workspace 3: Comms

## Shutdown Sequence
- `hyprctl dispatch exit`
- `systemctl --user stop waybar`
