# Platform Ecosystem

MaxOs is a custom desktop environment built on top of Hyprland and Fedora 43. It is designed to be a modern, minimal, and efficient desktop environment.

## Key Features

- Customized Hyprland/Fedora 43 desktop environment.
- Modern, easy-to-use, and efficient desktop environment.
- Efficient desktop environment.

- Fedora 43
