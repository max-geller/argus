# Quickstart

## Overview

The MaxOS Quickstart Guide is a comprehensive guide to help you get started with MaxOS, a customized Hyprland/Fedora 43 desktop environment.

## Key Concepts

The MaxOS platform was built with the following key concepts in mind:

- Provide comprehensive documentation for the platform using a modern, easy-to-use interface.
- Allow graphical customization of the platform.
- Develop custom widgets and tooling for the platform.
- Allow for easy extension of the platform.


## Prerequisites

In order to build and run the MaxOS platform, you will need to meetthe following prerequisites:

- Fedora 43
- Hyprland
- Rust
